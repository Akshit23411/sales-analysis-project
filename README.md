Sales Data Analysis and Visualization using Python

📌 Project Overview

This project focuses on analyzing and visualizing sales data using Python. The main objective was to perform data cleaning, exploratory data analysis (EDA), and generate meaningful business insights through visualizations and statistical analysis.

The project was completed as part of a Data Analytics Internship task.

---

🚀 Features Implemented

- Data Loading using CSV dataset
- Data Cleaning and Preprocessing
- Descriptive Statistical Analysis
- Univariate Analysis
- Bivariate Analysis
- Data Visualization
- Insight Extraction
- Dashboard Creation

---

🛠 Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Google Colab
- GitHub

---

📊 Visualizations Included

- Histogram
- Box Plot
- Correlation Heatmap
- Scatter Plot
- Bar Chart
- Pie Chart

---

📈 Key Insights

- Most sales values are concentrated in lower ranges.
- Sales and profit show a positive correlation.
- Some regions generated lower profitability.
- Discounts negatively impacted profit in several cases.
- Technology category contributed higher sales performance.
